Simple PHP BC Math library
==========================

A simple PHP BCMath library for the platforms which do not install the PHP BC Math module (version before PHP 4.0.4)

PHP BC Math module (http://php.net/manual/en/book.bc.php)

Supports
--------
* bcadd
* bcsub
